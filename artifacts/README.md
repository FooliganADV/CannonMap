# Planner validation artifacts

These files were generated by the browser-based planner validation script.

The current repository does not contain the official America 250 master or daily GPX files, so the checked-in GPX and QA artifacts use `tests/fixtures/rally-project.cmap`. They prove the export workflow but are **not rally-ready navigation files**. Re-run `scripts/planner-validation.mjs` with the official GPX files before accepting this branch for rally planning.
